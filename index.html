問題診斷與解決

測試 1: 檢查 Apps Script 權限

確保您的 Google Apps Script 已正確設置權限：

  1. 打開您的 Google Apps Script 項目
  2. 點擊"部署" > "部署設定"
  3. 確保"執行身份"設置為"我"
  4. 確保"誰有存取權"設置為"任何人"

測試 2: 檢查 Google Sheets 權限

確保您的 Google Sheets 文件已正確設置權限：

  1. 打開您的 Google Sheets 文件
  2. 點擊右上角的"共享"按鈕
  3. 確保"一般存取"設置為"知道連結的任何人"，並具有"編輯者"權限

測試 3: 驗證 Apps Script 程式碼

確保您的 Google Apps Script 程式碼正確無誤：

function doPost(e) {
  try {
    const data = JSON.parse(e.postData.contents);
    const sheet = SpreadsheetApp.getActiveSpreadsheet().getActiveSheet();
    
    if (data.action === "getRecipes") {
      return getRecipes(sheet);
    }
    
    if (data.action === "addRecipe") {
      return addRecipe(sheet, data);
    }
    
    return ContentService.createTextOutput(JSON.stringify({
      status: "error",
      message: "未知操作"
    })).setMimeType(ContentService.MimeType.JSON);
    
  } catch (error) {
    return ContentService.createTextOutput(JSON.stringify({
      status: "error",
      message: error.toString()
    })).setMimeType(ContentService.MimeType.JSON);
  }
}

function doGet(e) {
  return ContentService.createTextOutput(JSON.stringify({
    status: "error",
    message: "請使用POST請求"
  })).setMimeType(ContentService.MimeType.JSON);
}

function getRecipes(sheet) {
  const data = sheet.getDataRange().getValues();
  const recipes = {};
  
  for (let i = 1; i < data.length; i++) {
    const row = data[i];
    const title = row[0];
    
    if (!recipes[title]) {
      recipes[title] = {
        title: title,
        ingredients: [],
        steps: row[5] || "",
        timestamp: row[6] || ""
      };
    }
    
    recipes[title].ingredients.push({
      name: row[1] || "",
      weight: row[2] || 0,
      percent: row[3] || "",
      desc: row[4] || ""
    });
  }
  
  return ContentService.createTextOutput(JSON.stringify({
    status: "success",
    data: Object.values(recipes)
  })).setMimeType(ContentService.MimeType.JSON);
}

function addRecipe(sheet, data) {
  const timestamp = new Date().toISOString();
  
  data.ingredients.forEach(ingredient => {
    sheet.appendRow([
      data.title,
      ingredient.name,
      ingredient.weight,
      ingredient.percent,
      ingredient.desc || "",
      data.steps,
      timestamp
    ]);
  });
  
  return ContentService.createTextOutput(JSON.stringify({
    status: "success",
    message: "食譜已添加"
  })).setMimeType(ContentService.MimeType.JSON);
}

如何解決資料存儲問題

  1. 檢查 Apps Script 權限：確保您的 Apps Script 已正確部署並設置了適當的權限
  2. 檢查 Google Sheets 權限：確保您的 Google Sheets 文件允許任何人編輯
  3. 驗證程式碼：確保您的 Apps Script 程式碼正確無誤
  4. 檢查執行日誌：在 Apps Script 編輯器中查看執行日誌，檢查是否有錯誤訊息
  5. 重新部署：如果問題持續存在，嘗試重新部署您的 Apps Script
提示： 如果以上步驟都無法解決問題，請檢查瀏覽器的開發者工具（按F12）中的"網路"標籤，查看發送請求的詳細資訊和回應。
