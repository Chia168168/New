如何獲取 Web App URL

  1. 打開您的 Google Apps Script 項目
  2. 點擊"部署" > "新增部署"
  3. 選擇"網頁應用程式"
  4. 設置執行身份為"我"，權限為"任何人"
  5. 點擊"部署"
  6. 複製生成的 Web App URL
  7. 將 URL 貼到上面的輸入框中
  8. 點擊"保存 URL" 按鈕
提示： 保存 URL 後，系統會自動測試連接。如果測試成功，您就可以開始使用食譜管理功能了。
